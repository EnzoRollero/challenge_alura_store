# challenge_alura_store
Challenge Alura Store Latam practicando Python y Manejo de Datos

Este challenge trata de ayudar a una persona para poder decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento.
Para esto, se analizó datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. 
El objetivo principal fue identificar la tienda menos eficiente y presentar una recomendación final basada en los datos.

El challenge permitió la practica de:
- Cargar y manipular datos CSV con la biblioteca Pandas.
- Crear visualizaciones de datos con la biblioteca Matplotlib.
- Analizar métricas como ingresos, reseñas y rendimiento de ventas.

El archivo continene en una primera etapa un analisis de los datos, en segunda etapa la generacion de graficos para el informe y por ultimo el informe final resultante del analisis con la recomendacion accionable.
